# Exterior Brush Visual Reference

A visual guide to each brush in the Exterior Map Brush Set, showing their SF Symbol icons and describing their visual appearance.

---

## 🎨 BASIC BRUSHES (3)

### Pen
**Icon:** `pencil`  
**Appearance:** Clean, solid line with medium weight  
**Visual:** ━━━━━━━━━━━━  
**Best for:** General outlines, borders, notes

### Fine Pen
**Icon:** `pencil.line`  
**Appearance:** Thin, precise solid line  
**Visual:** ─────────────  
**Best for:** Details, labels, fine outlines

### Marker
**Icon:** `highlighter`  
**Appearance:** Wide, semi-transparent line  
**Visual:** ▬▬▬▬▬▬▬▬▬▬▬  
**Best for:** Area highlighting, broad strokes

## Technical Notes

### Width Ranges

| Size Class | Range | Examples |
|------------|-------|----------|
| Fine | 0.5-3pt | Fine Pen, Stream, Trail |
| Narrow | 2-8pt | Pen, Path, Building |
| Medium | 5-15pt | Road, Hills, Forest |
| Wide | 10-25pt | Highway, City, Plains |
| Very Wide | 20-50pt | Ocean, Desert |

### Opacity Ranges

| Opacity | Use Case | Examples |
|---------|----------|----------|
| 40-50% | Subtle overlay | Plains, Marker |
| 50-70% | Semi-transparent | Desert, Tundra, Grassland |
| 70-85% | Standard | Most brushes |
| 85-100% | Solid | Roads, Structures, Water |

## Platform Differences

### iOS/iPadOS (PencilKit)
- Full pressure sensitivity support
- Smooth tapering
- Native pattern rendering

### macOS (Custom Rendering)
- Mouse/trackpad input
- Simulated pressure from speed
- Custom pattern implementation

Both platforms support all brush features, with slight visual differences.

---

## Performance Tips

1. **Large Areas**: Use wide brushes (Plains, Ocean) for faster coverage
2. **Details**: Switch to fine brushes (Fine Pen, Trail) for precision
3. **Organic Look**: Use brushes with scatter (Forest, Mountains)
4. **Organized Look**: Use grid-snapping brushes (Structures, Farmland)
5. **Natural Flow**: Use high smoothing brushes (River, Stream)
