### 🗺️ Coastlines & Borders (4)

Define boundaries and edges.

| Brush | Width | Pattern | Scatter | Use Case |
|-------|-------|---------|---------|----------|
| **Coastline** | 4.0 | Solid | 0.8 | Land/water boundary |
| **Border** | 3.0 | Dashed | 0.6 | Political boundaries |
| **Cliff** | 6.0 | Hatched | 0.7 | Elevation drops |
| **Mountain Ridge** | 5.0 | Solid | 0.7 | Peak connections |

**Special Features:**
- All brushes have **high scatter** for organic, jagged appearance
- Border uses **red color** for visibility
- Cliff uses **hatching** to show direction with rotation variation
- Coastline has **low smoothing** (0.3) for irregular edges
- Mountain Ridge has **very low smoothing** (0.2) for rugged peaks

**Workflow Tips:**
1. Draw Coastline with irregular strokes for natural shorelines
2. Add Border for political divisions (scatter creates hand-drawn look)
3. Use Cliff for dramatic terrain changes (jagged edge effect)
4. Connect mountains with Mountain Ridge (irregular peaks)

**Why High Scatter?**
Natural features like coastlines, cliffs, and ridges are never perfectly smooth. The high scatter amount (0.6-0.8) combined with rotation variation creates realistic, organic edges that look hand-drawn rather than computer-generated.

## Special Features Guide

### 🎯 Scatter Amount
Creates randomness in stamp placement or line variation for organic appearance.
- **Very High** (0.7-0.8): Coastline, Cliff, Mountain Ridge (jagged, irregular)
- **High** (0.6-0.7): Jungle, Grassland, Border (organic, natural)
- **Medium** (0.3-0.4): Mountains, Forest, Villages (moderate variation)
- **Low** (0.2): Cities (more organized)
- **None** (0.0): Single Tree, Buildings (precise placement)

**Note:** Edge-defining brushes (coastlines, cliffs, ridges, borders) use high scatter to create realistic, hand-drawn irregular edges rather than smooth computer-generated lines.

### 🔄 Rotation Variation
Randomizes stamp orientation.
- **360°**: Trees (all directions)
- **15-20°**: Mountains (slight variation)
- **None**: Structures (keep upright)

### 📏 Size Variation
Randomizes stamp size for variety.
- **0.4**: Mountains, Jungle (very varied)
- **0.3**: Forest, Hills, Villages (moderate)
- **0.25**: Single Tree (slight variation)

### 📐 Snap to Grid
Aligns strokes to grid for organized placement.
- **Enabled**: All structures, Farmland
- **Disabled**: Natural features

### 🎭 Tapering
Gradually reduces width at stroke ends.
- **Both Ends**: Rivers, Streams (natural flow)
- **None**: Most other brushes

## Quick Tips

### Performance
- Use larger brush sizes for faster coverage
- Group similar features on the same layer
- Use lower scatter values for better performance

### Aesthetics
- Let brushes overlap slightly for organic blending
- Vary brush sizes for depth perception
- Use lighter opacity for distant features

### Organization
- Keep terrain on bottom layers
- Place structures on top layers
- Use annotations layer for all text

### Workflow Efficiency
- Start with largest features (oceans, mountains)
- Work from background to foreground
- Save frequently during complex work

## Common Issues & Solutions

### Brushes appear too large/small
- Adjust width slider in brush properties
- Check canvas zoom level
- Consider map scale

### Stamps appear too regular
- Increase scatter amount
- Enable rotation variation
- Vary size manually

### Roads don't connect smoothly
- Increase smoothing value
- Use fewer control points
- Enable snap to grid

### Structures not aligned
- Enable snap to grid
- Use grid overlay
- Manually adjust placement
